# GSS × LLM Bias Dashboard

A 50-year interactive lens on American public opinion, built as Phase 1 of an MSAI 490 practicum evaluating LLM bias.

**What it does.** Turns the General Social Survey (1972–2024, 75k respondents, 6.9k variables) into a shared coordinate system where LLM answers can later be plotted against historical human responses.

- **Overview.** 12 canonical opinion variables on a slope chart. Shows that the human baseline is a moving target, not a fixed answer.
- **Explorer.** Pick any GSS variable, see its 50-year response distribution.
- **Compare.** Pick two variables, pick a stance for each, see if they shift together.

**Note.** GSS is a US-only probability sample. Human baseline here means Americans, not humanity.

**Data.** [General Social Survey](https://gss.norc.org/) · [Data Explorer](https://gssdataexplorer.norc.org/)
